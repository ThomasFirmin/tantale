# Tantale algos

Concrete optimization algorithms for the Tantale AutoML library,
built on top of Tantale core.

Import this crate via the top-level `tantale` re-export:
```rust,ignore
use tantale::algos::{random_search, BatchRandomSearch, sha, Sha, asha, Asha, hyperband, Hyperband};
```
